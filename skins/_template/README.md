# Szablon skórki TTS Hub

1. Skopiuj ten folder do `%APPDATA%/TTS_hub/skins/<id>/` (Windows).
2. Zmień `id` w `skin.json` — musi być identyczny z nazwą folderu.
3. Dostosuj `tokens` (zobacz `docs/SKINS.md`).
4. W aplikacji: **Ustawienia → Wygląd** — wybierz skórkę lub **Importuj** paczkę `.ttskin`.

Zbuduj `.ttskin`: spakuj `skin.json`, `skin.css` i opcjonalnie `preview.png` do ZIP i zmień rozszerzenie na `.ttskin`.
