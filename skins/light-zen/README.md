# Light / Zen (szablon własnej skórki)

Wbudowana skórka o tym samym `id` jest w aplikacji (`src/skins/builtin/light-zen.*`).

Ten folder służy do:

- importu paczki `.ttskin` na innej maszynie,
- dalszej personalizacji (kolory, `skin.css`) bez przebudowy aplikacji.

**Instalacja:** Ustawienia → Wygląd → Importuj skórkę… (ZIP z tym folderem lub `.ttskin`).

Uwaga: jeśli `id` koliduje z wbudowanym `light-zen`, zmień `id` w `skin.json` i nazwę folderu (np. `light-zen-moja`).
