## Jak zaktualizować

1. Uruchom `npm run tauri dev`.
2. Ustaw okno na ~1400×900.
3. Zrób zrzut (Win+Shift+S) lub użyj skryptu capture z README głównego repo.
4. Zastąp pliki w tym folderze i zaktualizuj README w razie nowych widoków.

- Modal „Ustawienia zaawansowane”
- Zakładka Archiwum z kilkoma wpisami
